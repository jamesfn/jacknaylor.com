yt2mp3

yt2mp3 is a simple terminal-level program built with python. The program utilizes the pytube package to download audio directly from a YouTube video (excluding "Topic" channels) and saves as an mp3 file. This program is faster than any website accomplishing the same thing, and the process is even faster if you put the yt2mp3 file in the same directory as your downloads. Requires python3 and pytube to work.
